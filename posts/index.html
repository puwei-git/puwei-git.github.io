第一步：下载Pycharm安装包

  • 访问 Pycharm 官网，下载 Pycharm 2022.3 版本的安装包，下载链接如下 : https://www.jetbrains.com/pycharm/download/other.html，也可以直接使用笔者下载好分享出来的。
  • 打开页面后，在version 2022.3中选择2022.3.2，点击下载2022.3.2-Windows(exe)。

第二步：安装Pycharm 2022.3版本

  • 如果电脑之前有安装老版本 Pycharm， 需要先卸载。
  • 卸载完成后，双击刚刚下载好的 Pycharm 2022.3 版本安装包。弹框会提示选择安装路径，我这里直接选择的默认安装路径C:\Program Files\JetBrains\PyCharm 2022.3.2
  • 然后，勾选 Create Desktop Shortcut 创建 Pycharm 桌面快捷启动方式，以方便后面快速打开 Pycharm，再点击 Next 按钮 ，点击 Install 按钮，开始安装。
  • 等待安装完成后，勾选 Run Pycharm, 点击 Finish 按钮即运行 Pycharm 。 Pycharm 运行成功后，会弹出如下对话框
